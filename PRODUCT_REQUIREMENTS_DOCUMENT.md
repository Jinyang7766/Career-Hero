# Career Hero - 产品需求文档 (PRD)

## 1. 产品概述 (Product Overview)
**Career Hero** 是一款基于人工智能的求职辅助平台。通过集成的 Google Gemini 3.0 大模型和 Vector RAG 技术，为用户提供从简历解析、智能润色、多维度诊断到沉浸式面试模拟的全链路求职加速服务。

## 2. 产品目标 (Project Objectives)
*   **提升简历竞争力**：通过 AI 深度分析及行业标杆对齐，纠正简历中的薄弱环节。
*   **消除面试焦虑**：提供高度真实的面试环境，帮助用户在实战前充分练习。
*   **实现人岗精准匹配**：结合具体职位的 JD (Job Description)，提供定制化的简历优化与面试题库。
*   **高效管理求职资产**：提供一站式的简历管理、预览及导出功能。

## 3. 用户画像 (User Personas)
*   **应届毕业生**：缺乏简历撰写经验，需要专业的结构化引导和面试入场券。
*   **职场跳槽者**：希望针对特定大厂或行业（金融、互联网等）进行简历深度定制化。
*   **跨行业转型者**：需要 AI 帮助挖掘过往经历中的可迁移技能，并进行专业化转述。

## 4. 核心功能需求 (Functional Requirements)

### 4.1 简历管理与解析 (Resume Management)
*   **多源导入**：支持上传本地文件 (PDF, Word, Image) 或手动输入。
*   **AI 反向工程解析**：自动将文件内容提取并结构化为：个人信息、工作经验、项目经历、教育背景及技能栈。
*   **实时编辑器**：提供具有即时校验功能的编辑器，支持多模块灵活排序。
*   **版本控制**：支持针对同一份基准简历生成多个针对不同 JD 的优化版本。

### 4.2 AI 深度诊断 (AI Diagnosis)
*   **多维打分系统**：基于经历完整度、技能匹配度、格式规范性及语言专业性进行综合评分。
*   **缺陷检测**：自动识别描述模糊、缺乏量化成果、关键词缺失等常见问题。
*   **PIP (发现-改进-练习) 建议**：
    *   不仅告知“哪里不好”，还提供“如何改写”及对应的“面试应对建议”。
*   **PII 数据脱敏**：在调用 AI 接口前自动脱敏用户敏感信息（电话、邮箱等），确保数据安全。

### 4.3 Vector RAG 行业增强 (Vector RAG Enhancement)
*   **行业标杆对齐**：通过 Supabase `pgvector` 检索行业典型成功案例。
*   **硬核行业定制**：针对技术、金融、供应链等对关键词极其敏感的行业，引入行业知识库进行语义修正。
*   **技能缺口分析**：对比当前简历与 JD 要求，精准指出并填充核心技能关键词。

### 4.4 沉浸式 AI 面试 (AI Interview Simulation)
*   **全链路交互**：支持从“自我介绍”到“反向提问”的完整面试流程。
*   **多模态支持**：支持流式文本交互及低延迟的语音转文字 (STT) 录入。
*   **灵活性**：用户可随时选择“跳过此题”或“结束面试”。
*   **热身机制**：秒级启动第一道面试题，后续逻辑后台静默加载。

### 4.5 专业报告与多维导出 (Reporting & Export)
*   **商务化面试报告**：生成包含面试表现总结、亮点分析、待提升项及后续训练计划的结构化报表。
*   **离线资产生成**：
    *   **后端驱动 PDF**：基于 Playwright 生成像素级还原的商务简历 PDF。
    *   **前端驱动长图**：支持将面试分析报告导出为高清超长图，便于保存和查阅。
*   **多模板预览**：内置 Modern (商务蓝/现代感)、Classic (传统黑白/稳重) 及 Minimal (极简线框) 三套风格。

## 5. 关键业务流程 (User Flows)

### 5.1 基础路径 (Standard Path)
1. 注册/登录 -> 2. 创建或上传简历 -> 3. 输入目标公司及 JD -> 4. 启动 AI 诊断 -> 5. 接受优化建议并编辑 -> 6. 导出 PDF 投递。

### 5.2 进阶路径 (Interview Path)
1. 完成诊断 -> 2. 进入 AI 面试 -> 3. 完成 5-8 轮深度对话 -> 4. 生成面试分析报告 -> 5. 根据报告中的建议进行针对性复次。

## 6. 技术架构 (Technical Architecture)
*   **前端**：React 18 + TypeScript + Vite + Tailwind CSS (极致 UI 体验/响应式)。
*   **状态管理**：Zustand (全局状态) + React Context (UI 交互状态)。
*   **后端**：Flask + Python 3.12 (异步 gevent 模式支撑长连接)。
*   **AI 引擎**：Google Gemini 3.0 全家桶 (含 Flash 解析、Pro 诊断)。
*   **数据库**：Supabase (PostgreSQL + pgvector)。
*   **缓存与持久化**：本地 LocalStorage 与数据库混合存储，确保离线后会话可恢复。

## 7. 非功能性需求 (Non-functional Requirements)
*   **响应速度**：核心 AI 解析需在 3-5 秒内完成，面试流式响应延迟 < 200ms。
*   **安全性**：认证基于 JWT，核心资产（简历、报告）在数据库层面实现物理隔离。
*   **跨端适配**：全面适配移动端（Webview/H5），针对小屏幕进行字体缩放与布局压榨。

## 8. 未来规划 (Roadmap)
*   **多语言简历翻译**：支持中英日韩等主流求职语言的一键出海。
*   **面试录音回放**：记录面试全程语音，支持回听与语气分析。
*   **岗位自动投递助手**：集成主流招聘平台 API，实现一键投递优化后的版本。
